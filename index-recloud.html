Re-cloudchallenges

“Enabling Happiness and Productivity at Home #StayAtHome”

#Re-Cloudchallenges adalah sebuah kompetisi yang ditujukan kepada seluruh pelajar, lulusan dan developer di Indonesia sebagai platform untuk mengembangkan ide kreatif dan mempublikasikan ide tersebut menjadi sebuah situs website.

Jadilah juara!
Diselenggarakan & diadakan oleh:

Jadilah juara!

Ikuti event #Re-Cloudchallenges dan menangkan hadiahnya

  • Juara 1: $2000 + credit di Alibaba Cloud ($2000)
  • Juara 2: $1500 + credit di Alibaba Cloud ($2000)
  • Juara 3: $1000 + credit di Alibaba Cloud ($2000)
  • Juara 4: $800 + credit di Alibaba Cloud ($2000)
  • Juara 5: $500 + credit di Alibaba Cloud ($2000)
Cara Daftar

Proses Mendaftar

  • Langkah #1

    Mempunyai akun di CodePolitan https://www.codepolitan.com
  • Langkah #2

    Mempunyai akun di Alibaba Cloud https://www.alibabacloud.com
  • Langkah #3

    Mengisi Formulir Pendaftaran

Setelah mengisi formulir pendaftaran, tim codepolitan akan memverifikasi akun peserta di Codepolitan dan akun di Alibaba Cloud. Jika sudah diverifikasi, peserta akan mempunyai dashboard khusus di web Codepolitan.

Ketentuan Peserta

  • Peserta adalah pelajar, mahasiswa, lulusan, pekerja profesional (freelancer, developer, karyawan, dosen dll) dengan status kewarganegaraan WNI
  • Kompetisi ini bisa diikuti secara perorangan maupun tim, tetapi hanya satu situs web pemenang yang akan dihitung untuk setiap orang yang dikirim.

Ketentuan Kompetisi

  • Buat suatu produk web yg bertema (Bisa dilakukan secara individual maupun tim)
  • Produk dapat dibangun dengan perkembangan teknologi apa pun yang berlaku.
  • Situs web yang dikirimkan harus merupakan situs web yang berfungsi.
  • Hak cipta tetap menjadi milik pengembang (developer).
  • Situs web harus dipublikasikan di Alibaba Cloud Platform (mis : ECS)
  • Website harus dihosting oleh Alibaba Cloud
  • Setelah mempublikasikan, disarankan untuk mengatur domain untuk mengaksesnya, tetapi alamat IP sudah cukup.
  • Peserta dapat mengirimkan beberapa website, baik website baru maupun yang sudah ada.

Info Submission

Mengakses Re-Cloud challenges Submission Page on Codepolitan platform [link will provide].

Mengisi data:

  • Name
  • Acount ID
  • Website Link on AlibabaCloud: [Domain/IP Address]
Berikan dokumentasi dengan isi:

Ada empat kriteria yang harus dinilai oleh juri yaitu;

  • Menjelaskan Latar belakang Ide
  • Menjelaskan Masalah yang ditemukan
  • Menjelaskan Solusi yang diberikan
  • Menjelaskan Teknologi dalam produk yang digunakan
  • Link Video demo
  • Video/Link Video Demo Solusi
  • (opsional) Pembelian/ izin penggunaan / lisensi asset atau karya dengan hak cipta
Menggunakan stack lebih dari web jadi nilai tambah

Menggunakan stack lebih dari web jadi nilai tambah

Website yang sudah memiliki pengguna

Website yang sudah memiliki pengguna (dibuktikan dengan screenshot store atau screenshot dashboard analytics) akan mendapatkan nilai plus

Detail Penilaian

Twibbon Point (@ 100 poin)

Peserta yang membagikan gambar twibbon #AlibabaCoudchallenges di media sosial (Instagram Post/Facebook/Twitter) saat mendaftar akan mendapatkan 100 poin (twibbon bisa diakses di sini).

Judges Point (@ 2000 poin)

Ada empat kriteria yang harus dinilai oleh juri yaitu;

  • Functionality
  • Impact
  • Creativity & Originality
  • UI and UX
Poin maksimal setiap kriteria adalah 500 poin

Poin maksimal setiap kriteria adalah 500 poin

Voting situs web di media sosial (@ 400 poin)

Peserta dapat memposting tentang screenshot dan story website mereka di social media (Instagram Post/Facebook/Twitter) dan setiap like/retweet akan dikonversi menjadi 1 poin dengan poin maksimal yang didapat adalah 400.

Sertifikasi dari Alibaba Cloud (@ 1000 poin)

Peserta yang mengikuti ujian di AlibabaCloudEdu dan telah lulus akan memiliki 1000 poin.

Menghadiri semua Workshop dan Webinar (@ 500 poin)

Peserta yang mengikuti semua rangkaian Workshop dan Webinar akan mendapatkan 500 poin.

Detail Penjurian

  • Tim Juri akan langsung mendiskualifikasi / menggugurkan Solusi dengan tautan publik yang tidak dapat diakses.
  • Tim Juri dapat meminta penjelasan dan/atau meminta developer menunjukkan source code apabila diperlukan.
  • Tim Juri akan memeriksa Solusi setelah periode challenges selesai dan membutuhkan waktu 7-14 hari kerja untuk memeriksa dan mengevaluasi seluruh solusi.
  • Peserta diperkenankan memperbarui submission sebelum challenges berakhir.
  • Solusi akan gugur/diskualifikasi apabila peserta melakukan update setelah challenges berakhir, sampai diinformasikan pemenang challenges.
  • Peserta tidak diperkenankan mengambil asset, solusi, atau hak cipta pihak lain tanpa ijin atau lisensi yang jelas.
  • Peserta tidak diperkenankan kloning, reskin, dan menggunakan solusi dari pihak lain.
